Portfoli Page: The Game of Set Modified

The game of set is a math game used to let set theory. It is usually played by two or more players where cards are laid out in a 4 x 3 grid.This game of set is a mofified game of set. A traditional game of Set, has three combinations of four features. In this game hovever the there are only three combinations of three different features. These features include the color, shape, type of object. In this game I have replaced the traditional diamond, squiggly and oval with characters. These add a more game like look to the design of the game. In this game the player will play against a timer. If you have not form enough sets before the time runs out you failed the game, otherwise you win.

Rules for the games

  • A set consist of three cards in which the features are either the same on each card or different on each card
  • There can never be a set in which two items are the same and one is different
